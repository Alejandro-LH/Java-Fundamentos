# Java Fundamentos

Repositorio dedicado al aprendizaje y práctica de los fundamentos esenciales de Java.  
Este proyecto está enfocado en desarrollar las bases necesarias para comprender el lenguaje, resolver problemas de programación y construir aplicaciones simples utilizando programación orientada a objetos.

El objetivo principal es dominar los conceptos fundamentales que todo desarrollador Java debe conocer antes de avanzar a temas más avanzados o frameworks.

---

## Objetivos del repositorio

- Comprender la sintaxis básica del lenguaje Java
- Desarrollar lógica de programación usando estructuras de control
- Manipular datos utilizando arrays y colecciones
- Trabajar con cadenas de texto (Strings)
- Aplicar los principios básicos de la Programación Orientada a Objetos (OOP)
- Manejar errores mediante excepciones
- Prepararse para evaluaciones técnicas o entrevistas de nivel junior

---

## Contenido

### 1. Sintaxis básica

Fundamentos del lenguaje y estructura de un programa Java.

Temas:

- Estructura de un programa (`main`)
- Variables y tipos de datos
- Operadores aritméticos y lógicos
- Entrada y salida de datos (`Scanner`)
- Uso de `System.out.println`

---

### 2. Estructuras de control

Control del flujo de ejecución del programa.

Temas:

- Condicionales (`if`, `else`, `switch`)
- Ciclos (`for`, `while`)
- Control de flujo (`break`, `continue`)

---

### 3. Arrays y listas

Almacenamiento y manipulación de colecciones de datos.

Temas:

- Declaración y uso de arrays
- Recorrido de arrays
- Búsqueda y procesamiento de datos
- Uso de `ArrayList`
- Operaciones básicas sobre listas

---

### 4. Strings

Manipulación de texto en Java.

Temas:

- Métodos de `String`
- Comparación de cadenas
- Búsqueda de texto
- Conversión de tipos
- Recorrido de caracteres

---

### 5. Programación Orientada a Objetos (OOP)

Principios fundamentales para modelar objetos y comportamientos.

Temas:

- Clases y objetos
- Constructores
- Métodos
- Encapsulación
- Uso de `this`

---

### 6. Manejo de excepciones

Control de errores durante la ejecución del programa.

Temas:

- `try` y `catch`
- Tipos de excepciones
- Manejo de errores comunes
- Validación de datos

---

### 7. Collections

Uso básico del framework de colecciones de Java.

Temas:

- `ArrayList`
- Agregar y eliminar elementos
- Recorrer colecciones
- Verificar existencia de elementos

---

### 8. Lambdas y Streams (Introducción)

Conceptos básicos de programación funcional en Java.

Temas:

- Expresiones lambda
- Uso de `.stream()`
- Operaciones básicas (`filter`, `forEach`)
- Procesamiento de colecciones

---

## Requisitos

- Java JDK 17 o superior
- Editor o IDE (IntelliJ IDEA, VS Code, Eclipse)
- Conocimientos básicos de programación (recomendado)
